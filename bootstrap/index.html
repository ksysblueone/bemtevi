Ponto de partida

Uma visão geral do Bootstrap e conceitos básicos para iniciar um template padronizado.

Visão geral

Conheça as principais peças da arquitetura do Bootstrap, incluindo nossos metodos para melhorar, agilizar e fortalecer o desenvolvimento web.

HTML5 doctype

O doctype é obrigatório no início do documento para o funcionamento total do Bootstrap. Não necessariamente deva ter essa mesma estrutura abaixo, mas sua declaração é imprescindível.

<!DOCTYPE html>
<html lang="pt-BR">
  ...
</html>

Mobile first

O Bootstrap utiliza o conceito "Mobile First", ou seja, você desenha as telas, a interface primeiramente para versão mobile, como deve se comportar no celular, e após isso, para os demais dispositivos (tamanhos de telas).

Para o correto funcionamento do modo responsivo (principal característica do Bootstrap), deve-se ter certeza de que a meta tag viewport está definida e corretamente configurada entre as tags <head> do código.

<meta name="viewport" content="width=device-width, initial-scale=1">

Você pode desativar a ampliação do zoom em dispositivos móveis adicionando user-scalable=no na meta tag viewport. Isso desativa o zoom, ou seja, os usuários só são capazes de trabalhar com a barra de rolagem, e o resultado é que seu site ficará parecido com um aplicativo nativo. Entretanto, nós não recomendamos isso em todos os sites, por isso tome cuidado!

<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">

Containers

O Bootstrap requer um elemento que englobe o conteúdo das páginas, também para o funcionamento correto do seu sistema em grade. No Bemtevi utilizamos o .container-fluid e containers não podem ser aninhados.

Utilizando o .container-fluid ele ocupará toda a largura da página de acordo com o tamanho da tela e aplicará automaticamente as margens e enchimentos.

<div class="container-fluid">
  ...
</div>

Estrutura básica das páginas

Copie o HTML abaixo para iniciar uma página com o html mínimo aplicado com as classes básicas de um documento Bootstrap.

<div id="conteudo">
  <div class="container-fluid">
    <div class="row">
      <div class="col-md-12">
        <!-- Painel principal que engloba toda página -->
        <div class="panel panel-primary">
          <!-- Cabeçalho com o título da página atual -->
          <div class="panel-heading">
            <span class="panel-title">Título da página</span>
          </div>
          
          <!-- Conteúdo da página -->
          <div class="panel-body">
            <!--
            ...
            -->
          </div><!-- /.panel-body -->

          <!-- Rodapé da página com os botões de ação -->
          <div class="panel-footer">
            <button class="btn btn-primary" type="submit">Salvar</button>
            <a class="btn btn-link" href="javascript:history.back()">Voltar</a>
          </div><!-- /.panel-footer -->
        </div><!-- /.panel -->
      </div><!-- /.col-md-12 -->
    </div><!-- /.row -->
  </div><!-- /.container-fluid -->
</div><!-- /#conteudo -->
